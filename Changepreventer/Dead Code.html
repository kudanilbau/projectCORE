Dead Code

Dead code adalah kode dalam program yang tidak pernah dieksekusi saat program berjalan. Hal ini bisa terjadi karena kode tersebut tidak pernah dipanggil oleh bagian program lainnya, atau karena kondisi yang menyebabkan kode tersebut dieksekusi tidak pernah terpenuhi.

Dead code dapat menjadi masalah dalam pengembangan perangkat lunak karena dapat membuat program menjadi sulit dipahami dmn memperburuk performa program.

Dalam contoh ini, 'unused_function' , 'unused_variable, dan blok 'if False' adalah contoh dead code karena tidak ada bagian program utama yang memanggil atau mengaksesnya, sehingga mereka tidak akan mengaksesnya, sehingga mereka tidak akan pernah dieksekusi saat program berjalan.
