Parallel Inheritance Hierarchies


Penjelasan

Parallel Inheritance Hierarchies terjadi ketika terdapat dua hierarki kelas yang sejajar atau saling terkait dalam sistem. Kedua hierarki ini sering kali mengalami update yang terpisah, sehingga perubahan pada satu hierarki memerlukan perubahan yang sebanding atau serupa pada hierarki lainnya. Hal ini dapat menyebabkan code menjadi sulit dimengerti dan kompleks.


Faktor

Kurangnya pemisahan tanggung jawab dan ketergantungan yang berlebihan antara class.


Ada dua hirarki yang saling berhubungan hirarki 'Employee' dan 'Department'. Berikut adalah hirarki 'Employee'. 

        
public class Employee {
    private String name;
    private double salary;
    private String departmentName;

    // Constructor, getters, setters
}

public class Engineer extends Employee {
    private String role;

    // Constructor, getters, setters
}

public class Manager extends Employee {
    private int teamSize;

    // Constructor, getters, setters
}
