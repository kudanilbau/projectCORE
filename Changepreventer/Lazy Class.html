Lazy Class

Lazy class adalah salah satu "Code Smell yang mengindikasikan bahwa sebuah class tidak memiliki cukup tanggung jawab atau fungsionalitas yang cukupuntuk dapat membenarkan keberadaannya. Lazy class mungkin terjadi ketika sebuah class awalnya memiliki fungsionalitas yang signifikan, tetapi seiring waktu, sebagian besar fungsionalitasnya dihapus atau tidak lagi diperlukan, namun class tersebut tetap ada tanpa melakukan banyak hal.

Tanda-tanda dari Lazy Class antara lain :

1. Kelas memiliki sedikit atau bahkan tidak ada metode :

Jika sebuah class hanya memiliki beberapa metode dan tidak melakukan banyak hal, mungkin ituadalah tanda dari lazy class.

2. Kelas tidak digunakan secara aktif :

Jika class jarang digunakan atau tidak lagi digunakan sama sekali dalam kode, itu bisa menjadi indikasi bahwa class tersebut mungkin sudah tidak perlu lagi.

3. Kelas hanya digunakan untuk menyimpan data tanpa logika tambahan :

Jika class hanya digunakan untuk menyimpan data tanpa melakukan operasi atau logikatambahan pada data tersebut, itu bisa menjadi tanda lazy class.

Lazy class dapat mengurangi kejelasan dan kebersihan kode, karena mereka dapat menyebabkan kode menjadi lebih rumit dan sulit dipahami. Oleh karena itu, solusinya adalah menghapus atau menggabungkan dengan class lain untuk memperjelas struktur dan fungsionalitas kode.
